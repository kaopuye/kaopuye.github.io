存在注入可能，接下来，试一下单引号看看是否可以拼接。可以发现这关如果输入不符合sql语法是会在页面上返回报错信息的，根据这个就可以明确知道需要闭合什么符号，比如这关是闭合单引号（因为我们多加了一个引号，即就查询失败了）

这个 SQL 注入利用了单引号(')闭合前面的引号，然后用注释符(--+)注释掉后面的代码

  • -- : SQL 的单行注释标记

  • + : URL 中会被解码为空格

  • SQL 标准要求 -- 后必须跟空格或控制字符

通过以上结果确定是字符型注入，而且存在回显，可以使用联合注入
